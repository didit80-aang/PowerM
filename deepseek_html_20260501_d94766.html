📋 Skenario Demo Sidang TA

  1. Normal: Tunjukkan monitoring real-time semua parameter
  2. Overvoltage: Klik "Simulasikan Overvoltage" → Relay auto-trip
  3. Reset: Klik "Stop Simulasi" lalu "Reset" → Relay kembali ON
  4. Overcurrent: Klik "Simulasikan Overcurrent" → Relay auto-trip
  5. Manual Blocked: Coba klik "Manual" saat fault → Diblokir!
  6. Both Faults: Klik "Simulasikan Keduanya" → Multiple fault
  7. Mobile: Buka dashboard di smartphone → Tampilan responsif
  8. Data Logger: Tunjukkan riwayat fault tersimpan otomatis
